# 📝 To-Do List Web App

Una aplicación web de lista de tareas simple, intuitiva y con un diseño estético minimalista, 
desarrollada con el objetivo de dominar las bases del desarrollo web y la manipulación del DOM.
Su interfaz limpia y ligera está diseñada para eliminar distracciones, permitiéndote enfocar 
la atención en lo que realmente importa: organizar tus pendientes diarios de forma rápida y eficiente. 

Es un proyecto interactivo que permite añadir, marcar y eliminar tareas en tiempo real, 
combinando un código estructurado con una experiencia de usuario fluida y moderna.
